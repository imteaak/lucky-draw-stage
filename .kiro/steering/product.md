# Product Overview

Lucky Draw System ("Quay Số Trúng Thưởng") - A Vietnamese-language lottery/raffle application for events.

## Core Features
- Slot machine-style spinning animation to select random winners
- Import participant data via CSV or JSON
- Export winner history
- Admin panel for configuration and data management
- Audio feedback (BGM and sound effects)
- Session-based tracking with duplicate winner prevention

## User Flow
1. Import participant list (customer_code, prize_code, customer_name)
2. Click "QUAY NGAY" or press Space to spin
3. Slot machine animates and reveals winner
4. Save result or undo
5. Export winners when complete
